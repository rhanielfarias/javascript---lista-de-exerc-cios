#Lista de exercícios

##Exercícios Criação e Manipulação de Arrays:

* 1. Crie um array vazio. Use o método push para adicionar três números inteiros ao
     array.

* 2. Crie um array com alguns nomes de frutas. Use o método push para adicionar uma
     nova fruta ao final da lista.

* 3. Crie uma função que aceita um array e um elemento como argumentos e use o
     método push para adicionar o elemento ao array.

* 4. Crie um array com cinco elementos. Use o método pop para remover o último
     elemento e, em seguida, exiba o array resultante.

* 5. Crie um array com várias palavras. Use um loop for e o método pop para remover
     e exibir cada palavra uma por uma até que o array esteja vazio.

* 6. Crie uma função que aceita um array e use o método pop para remover e retornar
     o último elemento do array.

* 7. Crie um array com três números inteiros. Use o método shift para remover o
     primeiro elemento e, em seguida, exiba o array resultante.

* 8. Crie um array com nomes de cidades. Use um loop while e o método shift para
     remover e exibir cada cidade uma por uma até que o array esteja vazio.

* 9. Crie uma função que aceita um array e use o método shift para remover e
     retornar o primeiro elemento do array.

*10. Crie um array vazio e use o método unshift para adicionar três strings ao início do
     array.

*11. Crie um array com algumas cores. Use o método unshift para adicionar uma nova
     cor ao início da lista.

*12. Crie uma função que aceita um array e um elemento como argumentos e use o
     método unshift para adicionar o elemento ao início do array.

##Exercícios Iteração em Arrays:

* 1. Crie um array com nomes de pessoas. Use o método forEach para exibir cada
     nome no console.

* 2. Crie um array de números. Use o método forEach para multiplicar cada número por
     2 e exibir o resultado no console.

* 3. Crie uma função que aceita um array de números e usa o método forEach para
     calcular a soma de todos os elementos.

* 4. Crie um array com números inteiros. Use o método map para criar um novo array
     que contenha o quadrado de cada número.

* 5. Crie um array com nomes de animais. Use o método map para criar um novo array
     que contenha os nomes dos animais em maiúsculas.

* 6. Crie uma função que aceita um array de números e usa o método map para
     multiplicar cada número por um valor passado como argumento.

* 7. Crie um array com números inteiros. Use o método filter para criar um novo array
     que contenha apenas os números pares.

* 8. Crie um array com nomes de frutas. Use o método filter para criar um novo array
     que contenha apenas as frutas que começam com a letra "m".

* 9. Crie uma função que aceita um array de números e usa o método filter para
     retornar um novo array contendo apenas os números maiores do que um valor
     passado como argumento.

*10. Crie um array com números inteiros. Use o método reduce para calcular a soma de
     todos os elementos do array.

*11. Crie um array com palavras. Use o método reduce para criar uma única string que
     contenha todas as palavras concatenadas.

*12. Crie uma função que aceita um array de números e usa o método reduce para
     calcular a média dos valores no array.
